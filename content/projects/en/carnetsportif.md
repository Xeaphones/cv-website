## Objectives

Migration and upgrade of the CarnetSportif website, a platform focused on sports monitoring between coaches and athletes. This internship allowed me to confirm my web skills (especially ReactTS, Go and Figma) and work in a real team setup with a backend developer and a UI/UX designer.

## Key Elements

- Migration of the old website to ReactTS for a more dynamic rendering and a personalized experience for coaches and athletes.
- Use of Figma with a UI/UX designer to refresh the visual identity and improve accessibility, while making frontend integration smoother with prepared mockups.
- Use of a Go backend as an API server to deliver user data such as sessions, records and other performance information.

## Conclusion

My internship on the CarnetSportif migration was a very enriching experience. I confirmed knowledge I had built through personal projects, learned more about ReactTS and NodeJS libraries, and gained practical teamwork experience in a professional client-oriented context.
